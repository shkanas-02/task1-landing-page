# 🚀 Task 1 - Responsive Landing Page

This project is part of my internship at Elevate Labs.

## 📌 Objective

The goal of this task was to build a clean and responsive landing page using HTML and CSS. The page includes a header, hero section, and footer with modern UI design and mobile responsiveness.

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## ✨ Features

* Responsive design for mobile and desktop
* Hamburger menu for small screens
* Modern UI with clean color palette
* Hero section with call-to-action button
* Structured layout using Flexbox

Landing page designed for a digital brand with responsive layout and smooth user experienc)
